# 🔐 My Express Auth API

Đây là một RESTful API đơn giản sử dụng **Node.js + Express** với tính năng **authentication bằng JWT**.

---

## 🚀 Tính năng

- Đăng ký người dùng
- Đăng nhập và tạo JWT token
- Bảo vệ route với middleware xác thực token
- Mã hoá mật khẩu bằng bcrypt

---

## 🧰 Công nghệ

- Node.js
- Express
- JWT (`jsonwebtoken`)
- Bcrypt (`bcryptjs`)
- Dotenv
